# Colors

Colors which are used in the Contensis UI for [projects](project).

## Supported colors

The following list contains the colors that are currently supported:

- Blue
- Charcoal
- Cobalt
- Gray
- Green
- Orange
- Pink
- Red
- Ruby
- Teal
- Violet
- Yellow
